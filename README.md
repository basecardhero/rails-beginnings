# Rails Beginnings

A starter application to learn Ruby on Rails with React (webpack + yarn).

_Nothing fancy._

#### Goals

1. Learn [Ruby](https://www.ruby-lang.org/en/)
1. Learn [Ruby on Rails](https://rubyonrails.org/)
1. Learn [Webpack](https://webpack.js.org/)
1. Learn [React](https://react.dev/)
1. Learn [rspec](https://rspec.info/) for testing Rails
1. Learn [Jest](https://jestjs.io/) for testing javascript
1. Learn [Devise](https://github.com/heartcombo/devise) for authentication

#### Versions

- Ruby version `3.2.2`
- Node version `~18.12`
- Yarn version `~1.22`
- SQLite version `~3.39`

## Development

**0. Prerequisites**

Ensure you have the appropriate ruby, node, yarn, and sqlite versions installed.

**1. Clone this repository**
```
git clone git@github.com:basecardhero/rails-beginnings.git
```

**2. Move into the repository directory and install packages**
```
cd rails-beginnings
```

**3. Install gems**
```
bundle install
```

**4. Install javascript packages**
```
yarn
```

**5. Start the application**
```
bin/dev
```

Visit [localhost:3000](http://localhost:3000)

### Development Testing

**Rails tests**
```
bin/rspec
```

**Javascript tests**
```
yarn test
```
